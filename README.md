# Student Database Management System (SQL)

## Project Overview
This project demonstrates a basic Student Database Management System using SQL.
It performs CRUD operations (Create, Read, Update, Delete) on student records.

## Technologies Used
- SQL
- MySQL (or any relational database)

## Features
- Create database and table
- Insert student records
- View all records
- Search by roll number
- Update student marks
- Delete student record
- Calculate total, percentage, and grade using SQL queries

## Lanuage Used
- CREATE DATABASE
- CREATE TABLE
- INSERT
- SELECT
- UPDATE
- DELETE
- CASE statement
- Aggregate calculations
## How to run
1. Open MySQL or any SQL environment.
2. Copy the code from student_db.sql.
3. Execute the queries step by step.
